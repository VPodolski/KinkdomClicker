# Обзор экрана загрузки

Экран загрузки успешно внедрен! Теперь при запуске игры вы сначала будете попадать на этот экран, а затем — в само королевство.

## Что было сделано:
1. Сгенерирован великолепный пиксель-арт замка для баннера. Вы можете найти его в папке проекта `assets/startup_banner.png`.
2. Создана новая сцена `loading.tscn` и привязанный к ней скрипт `Loading.gd`.
3. В скрипте реализована логика:
   - Создается минималистичный интерфейс: темный фон, ваш новый баннер и полоса загрузки внизу.
   - Происходит фоновая загрузка основной сцены `main.tscn` через `ResourceLoader`.
   - Прогресс-бар плавно заполняется.
   - Добавлена искусственная задержка (около 2 секунд), чтобы экран не проскакивал за миллисекунду, и вы могли насладиться артом.
4. В файле `project.godot` стартовая сцена изменена на `loading.tscn`.

> [!TIP]
> Арт для баннера лежит по пути `assets/startup_banner.png`. Вы в любой момент можете заменить этот файл своей картинкой (с тем же именем), и она автоматически подтянется в игру!

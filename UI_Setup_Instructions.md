# Инструкция по сборке интерфейса для Карты Походов

Данный файл содержит пошаговую инструкцию по созданию визуальной части сцены `ExpeditionMapUI` в редакторе Godot 4. Все скрипты для логики уже написаны, осталось только собрать интерфейс и привязать правильные имена узлам (Nodes).

## Шаг 1: Создание новой сцены
1. В верхнем меню редактора нажмите **Scene -> New Scene** (или плюсик рядом со вкладками открытых сцен).
2. Выберите **User Interface** (создастся корневой узел `Control`).
3. Кликните дважды по имени созданного узла `Control` в панели Scene и переименуйте его в **`ExpeditionMap`**.
4. Сохраните сцену в папку `UI` под именем `ExpeditionMap.tscn` (сочетание клавиш `Ctrl+S`).

## Шаг 2: Привязка главного скрипта
1. Выделите корневой узел **`ExpeditionMap`**.
2. В панели инспектора (справа) или перетаскиванием из FileSystem привяжите к нему скрипт **`UI/ExpeditionMap.gd`**.

## Шаг 3: Контейнер для лагерей
1. Нажмите правой кнопкой мыши по узлу `ExpeditionMap` -> **Add Child Node** (Добавить дочерний узел).
2. Выберите обычный **`Control`**.
3. Переименуйте его строго в **`CampsContainer`** (это важно, скрипт ищет именно это имя).
4. (Опционально) Растяните его на весь экран, используя Anchors -> Full Rect.

## Шаг 4: Создание панели информации
1. Снова кликните ПКМ по корню `ExpeditionMap` -> **Add Child Node**.
2. Выберите **`Panel`** (Панель).
3. Переименуйте её в **`InfoPanel`**.
4. Настройте размер панели по своему вкусу (например, разместите её в правом нижнем углу или по центру экрана).

## Шаг 5: Наполнение панели информации
Теперь мы добавим текст и кнопки внутрь `InfoPanel`.
1. Кликните ПКМ по `InfoPanel` -> **Add Child Node**.
2. Выберите **`VBoxContainer`** (Вертикальный контейнер). Растяните его внутри панели (Anchors -> Full Rect).
3. Кликните ПКМ по `VBoxContainer` -> **Add Child Node** и добавьте следующие узлы строго в таком порядке:
   
   - **`Label`** (Текстовая метка)
     - Переименуйте её в **`PowerLabel`**.
     - В поле Text (справа) можно временно написать "Сила Врага: 0".
   
   - **`Label`**
     - Переименуйте её в **`StatusLabel`**.
     - Текст: "Статус: Ожидание".

   - **`Button`** (Кнопка)
     - Переименуйте в **`ScoutButton`**.
     - Текст: "Разведка".

   - **`Button`**
     - Переименуйте в **`AttackButton`**.
     - Текст: "В атаку!".

   - **`Button`**
     - Переименуйте в **`CloseButton`**.
     - Текст: "Закрыть".

## Шаг 6: Финальная проверка структуры
В панели Scene (слева) у вас должно получиться ровно следующее дерево:

```text
ExpeditionMap (скрипт ExpeditionMap.gd)
├── CampsContainer
└── InfoPanel
    └── VBoxContainer
        ├── PowerLabel
        ├── StatusLabel
        ├── ScoutButton
        ├── AttackButton
        └── CloseButton
```

*Обратите внимание: скрипт `ExpeditionMap.gd` использует пути через `$Name`, поэтому имена узлов должны в точности совпадать (регистр букв имеет значение).*

## Шаг 7: Добавление сцены в главную игру
1. Откройте вашу главную сцену `main.tscn`.
2. Найдите место, где у вас располагаются остальные экраны (UI зданий, войск и т.д., обычно это `TabContainer` или просто слои `CanvasLayer`).
3. Перетащите сцену `ExpeditionMap.tscn` из панели FileSystem прямо в ваше главное дерево интерфейса, чтобы добавить Карту Походов как новую вкладку или окно.

**Готово!** Теперь при запуске игры скрипт `ExpeditionManager` (который крутится в `Game.gd`) сгенерирует лагеря варваров, а `ExpeditionMap.gd` автоматически нарисует их в виде кнопок внутри `CampsContainer`. При нажатии на лагерь будет открываться `InfoPanel`.
